# Ranger Auto first-page concept

Next.js + TypeScript + Tailwind CSS + Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Behavior

- Sticky header stays fixed at the top.
- Only the header and left ABOUT rail are visible at first.
- Clicking ABOUT slides the "HEY EVERYONE!" card out to the right.
- Clicking the blank page overlay closes the card.
